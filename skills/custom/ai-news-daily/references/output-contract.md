# Output Contract

## File Path
- `${AI_NEWS_OBSIDIAN_DIR}/YYYY-MM-DD-ai-hotspots.md`

## Markdown Structure (strict)

```markdown
# 每日 AI 热点（YYYY-MM-DD）

## 1. 每日热点新闻（10条）
### 热点 01
- 标题：
- 来源：
- 发布时间：
- 链接：
- 一句话摘要：
- 热度分：

...（到 热点 10）

## 2. 话题选择（3 入选 + 7 未入选）
### 入选 01
- 话题：
- 入选原因：

...（到 入选 03）

### 未入选 01
- 话题：
- 未入选原因：

...（到 未入选 07）

## 3. 三篇话题推文（公众号短文）
### 文章 01
- 对应话题：
- 字数：

正文...

### 文章 02
...

### 文章 03
...
```

## Required Rules

- 热点数量必须为 10。
- 入选必须为 3，未入选必须为 7。
- 每篇文章 800-1200 字。
- 每条热点必须带链接。
- 输出语言统一为中文。

## Scoring Rules (High-Heat Priority)

总分 100：
- 来源权威性：35
- 时效性：25
- 跨源共现度：25
- 受众可写性：15

排序按总分降序。
